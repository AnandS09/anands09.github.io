Software Artifacts

SCALE-Sim

SCALE-Sim is a DNN accelerator simulator widely used in industry and academic research. SCALE-Sim allows users to get cycle accurate performance, memory accesses and (soon) energy consumption estimates for various dataflows in a systolic array based accelerator setting. Please visit the project webpage for details about the simulator and the tutorials to get started.
